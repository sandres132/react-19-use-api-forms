# React 19 + Nuevas APIs y Hooks

## Dev

1. Clonar el repositorio
2. Instalar dependencias

```
bun install
npm install
yarn install
```

3. Levantar API de planetas

```
bun run json-server
npm run json-server
yarn run json-server
```

4. Levantar el servidor de desarrollo

```
bun run dev
npm run dev
yarn run dev
```

5. Ver el resultado en el navegador

```
http://localhost:5173/
```
